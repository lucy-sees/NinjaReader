### Run Locally

To run **NinjaReader** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/lucy-sees/NinjaReader.git
```

Windows:

```bash
git clone https://github.com/lucy-sees/NinjaReader.git
```
